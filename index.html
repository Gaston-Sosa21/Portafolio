Proyectos

Panadería

Entrega final para las materias Programación III y Laboratorio de Computación III.

Se trata de la web de administración de una panadería. La plataforma permite la interacción de dos tipos de usuarios: Clientes, quienes pueden explorar productos y realizar pedidos, y Empleados, encargados de la gestión de pedidos y administración del sistema.

El desarrollo abarcó tanto el frontend como el backend, utilizando Microsoft Visual Studio como entorno de desarrollo y Microsoft SQL Server Management Studio para la administración de la base de datos, incluyendo la creación de tablas, registros y procedimientos almacenados.

Las tecnologías utilizadas incluyen C# con .NET Framework para la lógica de la aplicación y SQL server Management studio para la gestión de la base de datos.

Campus Virtual

Último proyecto y entrega final para la materia Laboratorio de Computación IV. Consistió en el desarrollo de una página web para una universidad. La aplicación está diseñada para dos tipos de usuarios: Docente y Administrador.

El desarrollo abarcó tanto el frontend como el backend, utilizando la IDE Eclipse junto con Java, HTML, CSS, y tecnologías como JSP y Servlets. Además, para la gestión de la base de datos, se empleó con MySQL Workbench.

Objetos perdidos

Este proyecto fue la entrega final de las materias Seminario y Programación Avanzada II, centrado en el desarrollo de una aplicación móvil con el objetivo de brindar apoyo social a sus usuarios.

La aplicación incorpora diversas funcionalidades diseñadas para facilitar la asistencia y comunicación entre quienes necesitan ayuda y quienes pueden ofrecerla.
